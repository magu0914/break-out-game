# Breakout — Portfolio Edition

https://magu0914.github.io/break-out-game/

学部の講義で作った簡素なブロック崩しを土台に、Web で遊べるよう整え直した小品です。


## 操作

| 操作 | キーボード / マウス | スマートフォン |
| --- | --- | --- |
| ラケット移動 | `←` `→` / `A` `D` / マウス | タッチして左右にドラッグ |
| ボール発射 | `Space` | 画面をタップ |
| ポーズ / 再開 | `Space` | (タップで再スタート) |
| リスタート | `R` | — |
| ミュート | `M` | — |

## 遊び方

- 5 行 × 8 列のブロックを全て崩すとステージクリア。
- ボールを落とすと残機が 1 減ります (初期値 3)。
- 残機ゼロでゲームオーバー。ハイスコアはブラウザに自動保存されます。
- スコア配分は上の行ほど高く `50 / 40 / 30 / 20 / 10` 点。2-hit ブロックは破壊時のスコアが 2 倍。

### ステージごとのギミック

| Stage | 名称 | 内容 |
| --- | --- | --- |
| 1 | CLASSIC | 標準の 5×8 グリッド |
| 2 | TOUGH ROW | 最上段が 2-hit (スチール色) |
| 3 | PYRAMID | ピラミッド型配置 |
| 4 | CORRIDORS | 通路のある配置 |
| 5 | CHECKER | 1-hit と 2-hit のチェッカー |
| 6 | FORTRESS | 外周と中央が 2-hit の要塞型 |
| 7+ | RANDOM | 12 種類のシェイプからランダム抽選。難易度に応じて 2-hit が増加 |

### パワーアップ (Stage 1 から約 22% でドロップ)

ブロック破壊時、カプセルがランダムに落下します。パドルで受け取ると効果発動。

| ラベル | 効果 | 立ち回り |
| --- | --- | --- |
| `W` WIDE   | パドル幅 1.6 倍 (15秒) | 安定枠。取りこぼし対策 |
| `P` PIERCE | ボール貫通 (8秒) | 縦に貫通させて一気に大量破壊 |
| `M` MULTI  | ボールを最大 5 個に分裂 | 同時破壊でスコア倍化 |
| `S` SLOW   | ボール減速 (10秒) | 立て直し用 |
| `L` +LIFE  | 残機 +1 (出現率低め) | 見かけたら最優先 |

ライフを失うと進行中の効果はすべてリセットされ、ボールは 1 個に戻ります。

## 技術スタック

- **p5.js 1.9** — Canvas 描画
- **Web Audio API** — オシレータ合成による効果音 (音声ファイル不要)
- **LocalStorage** — ハイスコア保存
- **Vanilla CSS** — モダンミニマルな UI
